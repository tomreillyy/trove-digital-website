Common questions

Everything you need to know before getting started.

Do I own the site?

Yes, if you choose the One-Off Custom option. You own all the code, assets, and design—forever. For the Monthly Website plan, the site stays live while you're subscribed. If you cancel, we can arrange a buyout.

What if I don't like the design?

We nail the design direction upfront with mockups before building anything. You'll approve the design before we write a single line of code. If changes are needed, we iterate quickly until you're happy.

Can I request changes after delivery?

Absolutely. Minor tweaks are usually included in the first week after delivery. For bigger changes, we can discuss scope and pricing—or you can add ongoing support if you need regular updates.

Do you write copy?

We can help refine and structure your copy to make it clearer and more compelling. If you need full copywriting from scratch, we can recommend trusted partners or discuss adding it to your project.

What happens if I cancel the monthly plan?

Your site comes offline. Simple as that. No lock-ins, no penalties. If you want to keep the site live, we can discuss a buyout price to transfer full ownership to you.
